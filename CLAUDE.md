# review-ai

## 하네스: AI 리뷰 작성기

**목표:** 사용자 입력(상품 정보 + 좋은점/개선점 + 태그)을 바탕으로 AI가 완성도 높은 쇼핑몰 리뷰를 생성하는 Next.js 앱을 구현한다.

**트리거:** 앱 구현, 컴포넌트 개발, API 통합, 기능 추가 요청 시 `orchestrate-review-ai` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
