## 서비스 내용
사용자가 리뷰를 간단히 작성하면 ai가 해당 내용을 참조하여 리뷰를 완성시키는 서비스

## 서비스 상세 내용
1. '쿠팡', '네이버' 등 리뷰를 작성할 플랫폼 선택
2. 상품 정보 입력
3. 직접 사용하면서 느낀 점을 좋은점, 개선점 input으로 직접 작성
4. 리뷰에 자주 사용되는 태그들이 보여지고 사용자가 선택할 수 있음
   - 예시: '가성비 있는', ''
5. 사용자가 선택한 내용들을 토대로 프롬프트를 구성하여 ai api로 보냄
6. ai가 작성한 리뷰를 받아서 사용자에게 보여줌

## 기능
- 플랫폼 선택 기능
- 상품 정보 입력(상품명, 카테고리)
- 좋은 점, 개선점 직접 작성 란
- 태그 선택 기능
- ai가 작성한 리뷰 간단하게 카피할 수 있는 버튼
- 이전 작성한 리뷰 보여주기 (일단은 로컬스토리지에 저장)
